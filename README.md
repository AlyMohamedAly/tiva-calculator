# Tiva Calculator
A Calculator that does simple arithmatic operations like addition, subtraction, multiplication, and division.

## Components
- Texas Instruments Tiva ARM® Cortex®-M4F Based MCU TM4C123G LaunchPad™
- LCD 16x2
- Keypad 4x4
- Potentiometer
- Resistors x4
- Jumpers
